使用 TypeScript 开发钻石矿工三消游戏。网格 7×7，包含普通宝石（红蓝绿黄紫）、泥土块（点击或消除相邻宝石可清除）、岩石障碍（不可交换，需使用道具消除）。玩家交换相邻宝石，消除三个以上得积分，被消除宝石上方的泥土块或矿石会下落一次。矿产（金、银、铜）随机埋在泥土之下，当泥土被清除后露出矿产，收集到矿产后计入进度。通关条件：达成目标分数（如 3000 分）和收集指定矿产数量（金:2，银:5，铜:8）且步数内（40 步）。添加道具：刷新棋盘、炸弹（消除 3×3）、鹤嘴锄（直接敲掉一个岩石）。连击加分。含 5 个闯关，难度递增。本地保存关卡进度和最高分。三套主题（矿洞、沙漠、地下城）。动画流畅，无卡顿。
